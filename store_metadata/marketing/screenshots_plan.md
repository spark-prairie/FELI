## Screenshot Specifications

### iOS (iPhone)
- **Dimensions:** 1242 x 2688 px (iPhone 13 Pro Max)
- **Format:** PNG or JPEG
- **Count:** Minimum 3, Maximum 10 (5 recommended)
- **Orientation:** Portrait
- **Device Frame:** Optional (recommended for clarity)

### Android
- **Dimensions:** 1080 x 1920 px (16:9 ratio)
- **Format:** PNG or JPEG (PNG preferred)
- **Count:** Minimum 2, Maximum 8 (5 recommended)
- **Orientation:** Portrait
- **Device Frame:** Optional

### Design Guidelines
- **Font:** San Francisco (iOS) / Roboto (Android)
- **Text Size:** Headlines 48-72pt, Subtext 24-36pt
- **Colors:** Match app brand (Purple #8B5CF6, Blue #3B82F6, Neutral grays)
- **Overlay:** Semi-transparent gradient overlay for text readability
- **Spacing:** Consistent padding (60px margins)

## Localization Notes

When creating localized versions of screenshots:

1. **Translate Headlines:** Use professional translation service, not machine translation
2. **Adjust Text Length:** Some languages (German, French) require more space
3. **RTL Languages:** Reverse layout for Arabic
4. **Cultural Sensitivity:** Ensure emojis and imagery are culturally appropriate
5. **Test on Device:** Verify legibility on actual devices

**Priority Languages for Localized Screenshots:**
1. English (Primary)
2. Spanish (Large market)
3. Chinese (Simplified) (Large market)
4. Japanese (High engagement)
5. French (EU compliance)
